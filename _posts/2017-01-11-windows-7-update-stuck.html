If Windows 7 updater is stuck and sucking up all of your memory make sure the following are installed manually:

  • Windows Update Agent 7.6.7.7600.320
  • KB3168965
  • KB3020369
  • KB3172605

Reboot windows before and after installing each update and before running the updater again
