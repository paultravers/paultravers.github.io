Windows 7 Update Stuck

Jan 11th, 2017 9:36 am

If Windows 7 updater is stuck and sucking up all of your memory make sure the following are installed manually:

  • Windows Update Agent 7.6.7.7600.320
  • KB3168965
  • KB3020369
  • KB3172605

Reboot windows before and after installing each update and before running the updater again

Upgrading Mongo From 3.x -> 3.2.1

Feb 11th, 2016 4:08 pm

Important note for sharded clusters!

3.2.1 has a bug in it for mongoc config metadata, if you upgrade there is a chance that you will run into this and need to downgrade your mongos client until there is a fix: BUG-SERVER-22247

Steps to upgrade:

  • Upgrade each shard (mongod)
  • upgrade the config servers starting with the last in the config string
  • upgrade all mongos instances

Slow Cooker Buffalo Shredded Chicken

Feb 11th, 2016 3:51 pm

Ingredients

  • 6 chicken breasts
  • 1 (1.5 ounce) packet of Simply Organic Ranch Seasoning/Dressing Mix
  • 1 (12 ounce) bottle Frank’s Hot Sauce

Instructions

  • Place chicken breasts in slow cooker. Sprinkle ranch seasoning over chicken then pour in hot sauce. Turn chicken a few times until it is coated and liquids and seasoning are mixed together.

  • Cook on low for 5-6 hours. Take chicken out and shred.
